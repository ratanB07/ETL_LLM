Prediction Endpoint
https://promote.alteryx.com/api/v1/models/customer-churn/predict
This endpoint accepts input data in JSON format and returns predictions with optional explanations and confidence scores.
Example Request
POST /api/v1/models/customer-churn/predict HTTP/1.1
Host: promote.alteryx.com
Content-Type: application/json
Authorization: Bearer <your_api_key>

{
  "data": [
    {
      "customer_id": "CUST12345",
      "tenure": 24,
      "monthly_charges": 79.99,
      "contract_type": "Month-to-month"
    }
  ],
  "params": {
    "include_explanations": true,
    "language": "en"
  }
}
